Teal API Document

# APIについて

Umiirosoft Teal の内部APIです。

APIの内容は予告なく変更されます。ご了承ください。

# APIの仕様

/tweet

GET: 全ての投稿を返します。

/tweet/:id

GET: idで絞り込んだ特定の投稿を返します。

/post

POST: 投稿を作成します。

クエリ
  • username
  • email
  • tweet

username: ユーザーの名前

email: ユーザーのメールアドレス

tweet: ユーザーの投稿内容

/delete/:id

GET/POST: idを指定して投稿を削除します。
